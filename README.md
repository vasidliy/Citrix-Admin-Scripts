# Citrix Admin Scripts

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A collection of PowerShell scripts, configuration examples, and utilities for managing **Citrix Virtual Apps & Desktops**, **NetScaler/ADC**, and related components (UPM, profiles, etc.).

## Contents

- [Profile Cleanup](./Profile-Cleanup/) – Remove old, corrupted, or orphaned user profiles (Citrix UPM, Microsoft Roaming) with quarantine, AD integration, and email reports.
- More scripts will be added over time (NetScaler backup, session monitoring, etc.)

## Usage

Each subfolder contains its own README with detailed instructions.
